## 🚀 Quick Start
1. Open Obsidian.
2. Select "Open folder as vault".
3. Point it to the root of this repository.
4. Enable the following plugins for the best experience:
	- **Dataview** (for dynamic queries)
	- **Templater** (for automated notes)
	- **Mermaid** (built-in, but ensure enabled)
	- **Canvas** (built-in)
